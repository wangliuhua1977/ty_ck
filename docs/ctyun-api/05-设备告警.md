# 设备告警（精简版）

## 1. 获取设备告警列表
接口：`/open/token/device/getDeviceAlarmMessage`

### 项目中的定位
只作为巡检和复核的辅助证据，不作为最终故障裁决。

### 常用参数
- `deviceCode`
- `alertTypeList`
- `startTime`
- `endTime`
- `pageNo`
- `pageSize`

### 常见告警类型
- `1`：设备离线
- `10`：设备上线
- `2`：画面变动 / 移动侦测
- `11`：有人移动

## 2. 当前项目规则
- 设备告警只能辅助说明问题，不能直接替代画面结论
- 复核页与通知页可展示最近告警摘要
