# 平台协议与鉴权（精简版）

## 1. 公共协议
- 协议：HTTPS
- 请求方式：POST
- Content-Type：`application/x-www-form-urlencoded`
- 请求头必须带：`apiVersion: 2.0`

## 2. 当前项目固定规则
- 公共参数 `version` 固定为 `1.1`
- 请求参数 `params` 使用 XXTea 加密
- 响应 `data` 统一按 RSA 私钥解密
- 告警推送同样按 RSA 解密

## 3. 公共参数
- `appId`
- `clientType`
- `params`
- `timestamp`
- `version`
- `signature`

## 4. 签名规则
签名按以下顺序拼接后做 HMAC-SHA256：
`appId + clientType + params + timestamp + version`

## 5. 获取访问令牌
接口：`/open/oauth/getAccessToken`

### 适用场景
当前项目使用：**用户无感知获取令牌**。

### grantType
- `vcp_189`：首次获取 accessToken
- `refresh_token`：刷新 accessToken

### 项目规则
- 启动时检查 token 是否可用
- 可用则继续复用
- 过期前/过期后走 refresh
- 不允许每个接口前重新申请 token

## 6. 项目实现建议
- 做统一平台网关，不允许业务层各自拼请求
- 做统一解密入口，不允许每个接口各写一套
- 把 token 缓存与刷新做成独立服务
