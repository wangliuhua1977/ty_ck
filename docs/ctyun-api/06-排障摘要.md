# 排障摘要（精简版）

## 1. 推荐固定规则
- `version` 固定使用 `1.1`
- 统一走 RSA 解密响应
- 统一维护一套解密路径

## 2. 常见问题方向
- 请求头缺少 `apiVersion: 2.0`
- appId / appSecret 不匹配
- token 过期
- IP 白名单问题
- 应用未开通接口权限
- 企业主参数错误
- 目录树权限不足
- RTSP 地址过期或播放器不兼容

## 3. 与当前项目最相关的排障点
### 加密解密
- 请求参数走 XXTea
- 响应 data 由 version 决定解密方式
- 当前项目固定使用 `1.1`，统一按 RSA 解密

### 视频流
- RTSP 地址不固定
- 地址可能很快失效
- 取到地址不等于能正常播放
- 播放失败不能立刻判设备故障

### 经纬度
- 平台坐标可能为空或不准
- 当前项目先原样同步入库，为地图预留
