# 天翼视联接口文档索引（项目精简版）

本目录只保留当前项目第一阶段真正需要的接口与规则，供 Codex 在开发时快速读取。

## 当前纳入的接口主题
1. 平台协议、鉴权、加密解密
2. 目录树与设备列表
3. 点位详情与状态
4. RTSP 直播地址与视频流信息
5. 设备告警
6. 排障摘要

## 当前项目的固定基线
- 公共请求参数固定使用 `version = "1.1"`
- 请求头固定带 `apiVersion: 2.0`
- token 采用本地缓存 + 过期刷新
- 点位详情主接口：`getDeviceInfoByDeviceCode`
- 视频主链路：RTSP
- 前台不用平台原始目录树，而用用户自定义目录
