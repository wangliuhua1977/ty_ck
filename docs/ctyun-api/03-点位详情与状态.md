# 点位详情与状态（精简版）

## 1. 点位详情主接口
接口：`/open/token/device/getDeviceInfoByDeviceCode`

### 当前项目中它的定位
这是点位详情的**主接口**。

### 主要字段
- `deviceCode`
- `deviceName`
- `deviceModel`
- `longitude`
- `latitude`
- `location`
- `isGbDevice`
- `gbId`
- `deviceType`
- `gat1400Id`
- `reportTime`

### 项目规则
- 平台经纬度必须入库
- 作为后续地图接入预留

## 2. 批量状态接口
接口：`/open/token/vpaas/device/batchDeviceStatus`

### 项目中的定位
只作为当前状态辅助信息，不作为最终故障裁决。

### 状态值
- `0`：离线
- `1`：在线
- `2`：休眠
- `3`：休眠（保活）

## 3. 单设备状态接口
接口：`/open/token/vpaas/device/getDeviceStatus`

### 使用场景
- 单点位详情页即时刷新
- 局部状态刷新

## 4. 项目规则
- 最终故障判断不以平台状态为准
- 平台状态只作为辅助手段展示和通知文案的一部分
